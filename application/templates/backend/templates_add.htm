模板维护
  • 模板分类：
  • echo form_error("main[t_type]","","");
  • 名称：
  • echo form_error("main[t_name]","","");
  • 模板htm:
    • 列表页
    • 内页
    • 样式1
    • 样式2
    • 样式3
    • 样式4
    echo form_error("main[t_html]","","");
    echo form_error("main[t_html_detail]","","");
  • （如：%name,%value，有几个字段就写几个）
  • 表格名称：
  • echo form_error("main[t_db_name]","","");
  • （如：infos as a）
  • 查询字段：
  • echo form_error("main[t_db_fields]","","");
  • （如：a.info_id,a.info_title,a.info_content,a.info_date）
  • 联合查询：
  • echo form_error("main[t_db_join]","","");
  • （如：admins as b,b.admin_id=a.info_user,left;admins as c,c.admin_id=a.info_user,left）
  • 查询条件：
  • echo form_error("main[t_db_where]","","");
  • 分组查询：
  • echo form_error("main[t_db_group]","","");
  • 排序：
  • echo form_error("main[t_db_order]","","");
  • （如：a.info_title asc,a.info_order desc）
  • 查询记录数：
  • echo form_error("main[t_db_limit]","","");
